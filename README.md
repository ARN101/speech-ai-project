# speech-ai-project

Bangla Automatic Speech Recognition (ASR) web application powered by a fine-tuned Moonshine model (`Sanji27/fountain`).

## Getting Started

1. Ensure you have `uv` installed.
2. Install Python 3.12.
3. Run the development server:
   ```bash
   uv run --python 3.12 uvicorn app:app --host 127.0.0.1 --port 8000
   ```
4. Access the web app at `http://127.0.0.1:8000`.
